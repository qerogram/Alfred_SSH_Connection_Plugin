1. We Apply a Workflow

2. Edit Action Command(IP, VMWare Local Address)

   (If, you don't have a VMWare Address, just you write a IP)

3. Set Environment(Workspace right click -> Open in Finder -> ALFRED_SSH_PLUGIN.py)
